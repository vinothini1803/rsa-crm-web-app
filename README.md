# RSA-CRA-WEB-APP

This RSA-CRA-WEB-APP  Web is for building a React application with Vite & Ant Design. Vite is a fast build tool that provides an instant dev server and optimized builds. Ant Design includes plenty of UI components to enrich web applications, improving component's experience consistently.

# Features

  - React 18.2.0
  - Vite 4.2.0
  - Less 4.1.3
  - Node 18.16.0
  - PrimeReact 9.2.2  
  - Fast development server with hot module replacement
  - Optimized production build
  - ESLint for code linting
  - Prettier for code formatting
  - CSS modules support

## Getting Started
To get started with this template, follow these steps:

1. Clone this repository: git clone [https://github.com/uitoux-solutions/RSA-CRM-Web-App.git]
2. Navigate to the project directory: cd RSA-CRM-Web-App
3. Install the dependencies: **npm install**
4. Start the development server: **npm run dev**
5. Open your browser and visit http://localhost:5173 to see the app in action.

## Available Scripts
In the project directory, you can run the following scripts:
- __npm run dev__: Starts the development server with hot module replacement.
- **npm run build**: Builds the app for production.
- **npm run serve**: Serves the production build locally.

## Folder Structure
The project structure is as follows:

```markdown
├── public
│   ├── vite.svg
├── src
│   ├── assets
│   │   ├── img
│   │   ├── styles
│   │   │   ├── theme.less
│   │   │   └── ...
│   ├── components
│   │   ├── common
│   │   └── layout
│   ├── pages
│   │   ├── masters
│   │   └── ...
│   └── main.jsx
├── index.html
├── vite.config.js
└── package.json
```

- The **public** folder contains the HTML template for the app.
- The **src** folder contains the application source code.
- The **components** folder is where you can place your React components.
- The **styles** folder is for CSS stylesheets.
- The **main.js** file is the entry point of the application.
- The **vite.config.js** file contains Vite configuration.
